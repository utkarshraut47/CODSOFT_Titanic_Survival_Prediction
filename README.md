Titanic Survival Prediction

This project uses the Titanic dataset to build a machine learning model that predicts whether a passenger survived or not. It is a classic beginner-friendly data science project that demonstrates the full workflow of a predictive modeling task, from data loading and exploration to preprocessing, model training, and evaluation.

The dataset contains important passenger information such as age, gender, ticket class, fare, family size, and embarkation port. In this project, missing values are handled, categorical variables are encoded, and a RandomForestClassifier is trained to make predictions. The model achieved an accuracy of 81.01% on the test set, showing strong predictive performance.

This project is a great example of how Python, pandas, matplotlib, and scikit-learn can be used together to solve a real-world classification problem.
